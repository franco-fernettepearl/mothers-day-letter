# 💌 Mother's Day Letter

A heartfelt, interactive Mother's Day letter built with HTML, CSS, and JavaScript - all in a single file.

## 📖 About

This project is a digital Mother's Day letter created as a personal tribute. It combines elegant design with interactive elements to present a meaningful message in a memorable way. The entire application runs in a single HTML file, making it easy to share and open on any device.
